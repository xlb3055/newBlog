# 力扣题解导航

按题型整理的 Java 题解，突出可复用思路和关键步骤。

## 题解固定结构

- 题意与限制
- 示例（输入/输出/解释）
- 解题思路（核心思想与步骤）
- Java 代码实现
- 复杂度分析
- 小结

## 题型目录

- 哈希
- 双指针
- 滑动窗口
- 子串
- 数组
- 矩阵
- 链表
- 二叉树
- 图论
- 贪心
- 堆
- 栈
- 二分查找
- 回溯
- 动态规划
- 多维动态规划
- 技巧

## 建议刷题顺序

1. 哈希 + 数组
2. 双指针 + 滑动窗口
3. 二分 + 栈 + 链表
4. 树 + 图
5. 贪心 + 堆
6. 动态规划 + 回溯

## 约定

- 默认语言：Java
- 推荐结构：入口类负责输入输出，解法类负责算法
- 输入若有简化会明确说明
