# 226. 翻转二叉树

---

## 题目
**描述：**
给你一棵二叉树的根节点 `root`，翻转这棵二叉树，并返回其根节点。

**示例：**
示例 1:
```
输入: root = [4,2,7,1,3,6,9]
输出: [4,7,2,9,6,3,1]
```
示例 2:
```
输入: root = [2,1,3]
输出: [2,3,1]
```
示例 3:
```
输入: root = []
输出: []
```

---

## 解题思路
### 核心方法
1.  **递归法**：利用树的递归结构，先递归翻转左子树和右子树，再交换当前节点的左右子节点，自底向上完成翻转。
2.  **迭代法**：使用广度优先搜索（BFS）或深度优先搜索（DFS）遍历树，在遍历过程中交换每个节点的左右子节点。

### 算法步骤（递归法）
1.  **终止条件**：当前节点为 `null` 时，直接返回 `null`。
2.  **递归翻转左子树**：递归处理左子树，得到翻转后的左子树。
3.  **递归翻转右子树**：递归处理右子树，得到翻转后的右子树。
4.  **交换左右子节点**：将当前节点的左子节点和右子节点交换。
5.  **返回当前节点**：返回翻转后的当前节点。

## 代码实现

### 递归法（Java）
```java
class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;
    TreeNode() {}
    TreeNode(int val) { this.val = val; }
    TreeNode(int val, TreeNode left, TreeNode right) {
        this.val = val;
        this.left = left;
        this.right = right;
    }
}

class Solution {
    public TreeNode invertTree(TreeNode root) {
        if (root == null) return null;
        TreeNode temp = root.left;
        root.left = invertTree(root.right);
        root.right = invertTree(temp);
        return root;
    }
}
```

### 迭代法（Java）
```java
import java.util.*;

class Solution {
    public TreeNode invertTree(TreeNode root) {
        if (root == null) return null;
        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);
        while (!queue.isEmpty()) {
            TreeNode curr = queue.poll();
            // 交换当前节点的左右子节点
            TreeNode temp = curr.left;
            curr.left = curr.right;
            curr.right = temp;
            // 继续处理子节点
            if (curr.left != null) queue.offer(curr.left);
            if (curr.right != null) queue.offer(curr.right);
        }
        return root;
    }
}
```

## 复杂度分析
### 递归法
- **时间复杂度**：`O(n)`，其中 `n` 是二叉树的节点数。每个节点会被访问一次。
- **空间复杂度**：`O(h)`，`h` 是二叉树的高度。递归栈的深度取决于树的高度，最坏情况下（链状树）为 `O(n)`。

### 迭代法
- **时间复杂度**：`O(n)`，每个节点会被入队和出队一次。
- **空间复杂度**：`O(m)`，`m` 是二叉树的最大宽度，最坏情况下（完全二叉树）为 `O(n)`。

---

## 总结
1.  递归法是最直观的实现方式，代码简洁，符合树的递归结构特性。
2.  迭代法通过队列或栈遍历树，避免了递归栈溢出的风险，适合处理深度较大的树。
3.  两种方法的时间效率一致，空间效率取决于树的形态，递归法受高度影响，迭代法受宽度影响。
