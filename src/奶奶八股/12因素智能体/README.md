# 12因素智能体（奶奶版）

这套内容来自 HumanLayer 的 **12-Factor Agents**，我用“奶奶也能看懂”的方式重写成了中文实战教程。

你可以把它理解成：

- 前半部分解决“怎么让智能体做对事”
- 中间部分解决“怎么让智能体能稳定跑”
- 后半部分解决“怎么让智能体可扩展、可运营、可维护”

## 阅读建议

1. 先看 `02` 和 `03`（提示词与上下文）
2. 再看 `05` 到 `09`（状态、控制流、错误恢复）
3. 最后看 `10` 到 `12`（架构升级）
