# 面试专题（Java 八股）

面向后端面试的系统化复盘，覆盖常见高频知识点与可复述答案。

## 覆盖范围

- Java 基础与语言细节
- 并发与线程协作、Java 内存模型、锁
- 集合与源码要点
- 计算机基础：Linux、操作系统、网络
- 数据库：MySQL、Redis、MongoDB、Elasticsearch、基础知识
- 杂碎记录：零散问题

## 建议顺序

1. Java 基础 + 集合
2. 并发与线程模型
3. 操作系统 + 网络
4. 数据库体系
5. 杂碎记录快速收尾

## 使用方式

- 侧边栏按专题浏览
- 每篇以问答式要点为主，适合快速复习
- 需要深挖时再回查官方文档或源码
